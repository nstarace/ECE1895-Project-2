## File Descriptions
All files relevant to the final bop-it product described and demonstrated are contained in one of three folders.
  
Hardware Design
* Cook-It: PCB design and manufacturing files
* Datasheets: Details on all components used in design, prototyping, and final product
* PCB BOM: Final bill of materials ordered for PCB assembly

Software Design
* Final: Lastest software revision running on the final product
* Triple LED Strip: Three LED zones populating colors and processing inputs from three sensors
* DFPlayer: Initial code for testing and configuring DFPlayback
* IMU_and_display_integration: Full functionality of "tilt" using the IMU to process inputs and increment score display
* Note: Other files present in this folder are used for support of the ones listed above.

Enclosure Design
* Cook-It Enclosure: Revision 1 of Easel design including STL files
* cookitv2: Revision 2 of Easel design including STL files
* Cookit sides and back: Revision 2 peripherals including STL files
